== Description ==

NOTE: The functionality this plugin provided is now build in to Wordpress! Happy times for all :)

The plugin adds additional buttons to the Wordpress Customizer Menu. These are used to test your website on various screen sizes.

**Features:**

* Test your website on various screen sizes within the Wordpress Customizer

**Included languages:**

* English

== Installation ==

1. Download the plugin and unzip it.
1. Upload the folder moosch-responsive-tester/ to your /wp-content/plugins/ folder.
1. Activate the plugin from your WordPress admin panel.
1. Enjoy!
